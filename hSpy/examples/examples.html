    What is the hSpy?

    hSpy - this is what you've been looking for. What you all this time, I could not find or did not have time to write.

    This plugin will help you to link your items on the menu with blocks on the page and switch between them during the transition to the desired unit.

    How to use it?

    jQuery("#menu > li").hSpy(

    "body > section", {

    classTargets : "active",

    classTitles : "current active",

    spaces : false,

    middleLine : 0,

    bottomLine : 0,

    axis : "y"

    }, {

    onBeforeSpace : function(){ ... }

    onAfterSpace : function(){ ... }

    onRealToggle : function(){ ... }

    onAlwToggle : function(){ ... }

    }

    jQuery($menuItems).hSpy(

    $sections, //Selector, Element, Array or jQuery

    $options, //Object options

    $hundlers //Object hundlers

    );

    Documentation

    Options

    spaces Boolean
    If False, then it is not visible when not 1 unit stands next
    classTitles String
    Classes established active title
    ClassTargets String
    Classes established active target
    middleLine Number/Function
    The distance from the top/left of the window, below which is considered to be a dedicated unit
    bottomLine Number/Function
    The distance from the bottom/right of the window, above which is considered to be a dedicated unit
    axis "x"/"y"
    Axis

    Hundlers

    onBeforeSpace function(prevData){...}
    It triggered when moving from Space to the Block
    onAfterSpace function(newData){...}
    It triggered when moving from Block to a Space
    onAlwToggle function(prevData, newTarget){...}
    Triggered when change target
    onRealToggle function(prevData, newTarget){...}
    Triggered when change the target with anything on the Block
